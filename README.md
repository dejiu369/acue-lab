# ACUE-Lab
Artificial Constraint Universe Engine

## Overview
ACUE is a unified framework that models existence as admissibility under constraints.

Instead of asking:
> What exists?

We ask:
> What is admissible?

## Core Idea
A system is defined as:

S = (X, O, V)

- X: state space
- O: transition operator
- V: admissibility constraint

Existence = admissibility.
